# Convention 

## Naming 

- camelCase:
  - variables,
  - functions,
  - object keys,
  - method names in classes.

- PascalCase:
  - types,
  - classes,
  - interfaces,
  - constructor functions,
  - component files,
  - component test files,
  - component style files,
  - component stories files,
  - component functions/classes,

- kebab-case:
  - files,
  - folders,
  - URL slugs,
  - CSS classes.

- UPPER_CASE_SNAKE_CASE
  - constants,
  - environment variables.
